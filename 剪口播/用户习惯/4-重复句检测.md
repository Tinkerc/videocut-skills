# 重复句检测

## 定义

相邻句子（被静音分隔）开头相同 ≥5字。

## 检测

```javascript
if (curr.text.slice(0, 5) === next.text.slice(0, 5)) {
  // 删较短的
  const shorter = curr.text.length <= next.text.length ? curr : next;
  markAsError(shorter);
}
```

## 案例

| 前句 | 后句 | 删除 |
|------|------|------|
| 这个工具 | 这个工具可以帮你 | 前句 |
| 我用cloud... | 我用cloud code | 前句 |
| 大家好 | 大家好 | 前句 |
