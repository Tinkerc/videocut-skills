# 用户习惯

用户个人偏好，AI 审核时参考。

## 文件清单

| 文件 | 类型 | 内容 |
|------|------|------|
| 1-核心原则.md | 原则 | 删前保后 |
| 2-语气词检测.md | 偏好 | 嗯啊呃 + 删除边界 |
| 3-静音段处理.md | 阈值 | >0.3s 删除 |
| 4-重复句检测.md | 偏好 | 相邻句子开头≥5字相同 |
| 5-卡顿词.md | 偏好 | 那个那个、就是就是 |
| 6-句内重复检测.md | 偏好 | A+中间+A 模式 |
| 7-连续语气词.md | 偏好 | 嗯啊、啊呃 |
| 8-重说纠正.md | 偏好 | 部分重复、否定纠正、词被打断 |

## AI 审核顺序

1. 静音 >0.3s → 必删（≤0.3s 自然停顿忽略）
2. 重复句 → 删短的
3. 句内重复 → 删 A+中间
4. 卡顿词 → 删前面
5. 重说纠正 → 删前面（部分重复、否定纠正、词被打断）
6. 语气词 → 人工确认
